📚 Coaching Website (MERN Stack)

A full-stack Coaching Management System built using MongoDB, Express.js, React.js, and Node.js (MERN).
This platform allows students to explore courses, enroll, and allows admins to manage courses, students, and analytics through a dashboard.

⚙️ Installation & Setup
1. Clone repository
git clone https://github.com/your-username/coaching-website.git
2. Backend setup
cd backend
npm install
npm start

Create .env file:

MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
3. Frontend setup
cd frontend
npm install
npm run dev
